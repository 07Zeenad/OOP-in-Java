# OOP in Java

This repository contains Java programs demonstrating the main OOP concepts:

## Concepts Covered

1. **Encapsulation**
   - Example: `src/encapsulation/Car.java` and `Main.java`

2. **Inheritance**
   - Example: `src/inheritance/Vehicle.java`, `Car.java`, and `Main1.java`

3. **Polymorphism**
   - Example: `src/polymorphism/Calculator.java`, `Animals.java` , `Dog.java` , and `Main.java`

4. **Abstraction**
   - Example: `src/abstraction/BankAccount.java` , `SavingsAccount.java` and `Main.java`
